flex布局

发表于 2020-02-01 更新于 2020-02-03 分类于 web前端

关于flex布局详细

1、首先，看图了解flex布局

2、flex容器相关设置

  • flex-direction 设置元素的排列方向
  • flex-wrap 在排列不下情况下，设置元素是否换行
  • flex-flow flex-direction和flex-wrap的简写
  • justify-content 元素在主轴上的对齐方式（受flex-direction影响）
  • align-items 元素在交叉轴上的对齐方式（受flex-direction影响）

3、flex元素相关设置

  • flex-grow 当有多余空间时，元素的放大比例
  • flex-shrink 当空间不足时，元素的缩小比例
  • flex-basic 元素在主轴上占据的空间
  • flex flex-grow、flex-shrink和flex-basic的简写
  • order 定义元素的排列顺序
  • align-self 定义元素自身的对齐方式

wx:if与hidden使用情况

发表于 2020-01-13 分类于 web前端

wx:if与hidden的区别与使用场景

区别：

  • wx:if会触发页面或组件重新加载；hidden只是页面显示隐藏来回切换
  • wx:if在页面和组件中使用都生效；hidden在页面中使用生效，组件中是不会直接生效，可以通过给组件添加属性方式实现

使用场景：

  • 频繁切换效果使用hidden，否则使用wx:if比较节省性能损耗
  • 组件中定义hidden属性
