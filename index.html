ES5特性总结

JSON对象

1、 JS(数组/对象)转为JSON(数组/对象)使用JSON.stringify()
2、JSON(数组/对象)转为JS(数组/对象)使用JSON.parse()

Array数组

var arr = new Array();

1、arr.length：数组长度
2、arr.indexOf(value)：使用数组中没有查到值返回-1，判断值是否存在数组中
3、arr.forEach(function(item, index){})：一般遍历数组
4、arr.map(function(item, index){})：遍历数组返回一个新的数组
5、arr.filter(function(item, index){ })：遍历且过滤出一个新的数组

Object对象

var obj = { name: ‘a’, age: 13 };

Object.defineProperties(obj, param)：为指定对象定义扩展多个属性

function方法

var obj = new Object();
var obj2 = new Object();

1、obj.call(obj2, arg1, arg2,…)
2、obj.apply(obj2, [arg1, arg2,…])
3、obj.bind(obj2, arg1, arg2,…)

相同点：
①、改变this指向的对象，第一个参数都为指向的原型对象

区别：
①、call()、bind()从第二个参数开始传形参，以此传入；
②、apply()第二参数必须为数组，数组为传递的形参。
、bind()、apply()会立即执行当前函数；bind()不会立即执行当前函数。
