发表于 2019-11-08 分类于 web前端

关于vue-cli打包语法检测

1、如果代码不符合ESlint标准、编辑器设置编辑规范、自动补全等；那么，你可能遇到的打包报错：

  • 完整js语句以;结束
  • 多余空格
  • 不需要解析的双引号改为单引号
  • 换行
  • 不全等比较改为全等比较（执行语法检测不会修改，手动改）
  • 方法名与小括号与花括号都要有空格
    等等…

2、如果使用vue-cli3创建项目时，没有选择Linter / Formatter（ESLint）标准检测或者写的代码不规范；那么，打包前要要执行下语法检测，再打包
