使用HBuilderX实现vue项目打包App

1、打包开发的vue项目（根据自己项目打包配置进行操作）

打包生成目录：

2、使用HBuilderX创建5+App项目
创建生成目录：

  • css 样式文件目录
  • img 图片资源文件目录
  • js JavaScript文件目录
  • unpackage 打包App资源目录 （通过 发行->原生App本地打包->生成本地打包App资源 来生成目录）
  • index.html 入口文件
  • manifest.json 打包App的配置文件（详细配置勾选参考详细说明）

3、构建打包app资源

  • 删除创建5+App项目目录下的css、img、js、index.html文件或目录
  • 拷贝打包的vue项目代码（dist目录下的文件）到 创建5+App项目目录

4、打包

  • 发行 -> 原生App-云打包
  • 详细配置勾选参考详细说明
