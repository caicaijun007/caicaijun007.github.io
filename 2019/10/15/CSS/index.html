总结CSS

发表于 2019-10-15 更新于 2019-10-21 分类于 web前端

CSS全称：Cascading Style Sheet（层叠样式表）

CSS特性

  • 继承性（子元素继承父元素的某些样式属性；不是所有属性都具有继承性；例如：padding、margin、border不具备继承性）

  • 层叠行（样式的覆盖；元素、属性、权重都相同，后者覆盖前者）

CSS优先级

行内样式>ID选择器>类选择器>标签选择器

不管是单个选择器还是多个选择器组合他们之间的优先级都可以通过上面这个公式来进行计算。我们将其称之为权重值。权重值越大，表示优先级越高。

样式冲突情况：

  • 引用方式冲突
  • 继承方式冲突
  • 指定样式冲突
  • 继承样式与指定样式冲突
  • ！important

CSS盒子模型

理解盒子模型：

  • 理解单独一个盒子的内部结构
  • 理解多个盒子之间的相互关系

盒子模型组成：

  • content（内容）
  • margin（外边距）
  • padding（内边距）
  • border（边框）
  • width（宽）
  • height（高）

每个元素都可以看成一个盒子

1、内容区：width和height属性指定盒子内容区的宽度、高度；overflow属性是隐藏溢出的内容。
2、外边距：两个盒子之间的距离；子元素与父元素之间的距离；兄弟元素之间的距离；margin、margin-top、margin-bottom、margin-left、margin-right。

3、内边距：内容与边框之间的空间；padding、padding-top、padding-bottom、padding-left、padding-right。

4、边框：border、border-width、border-style、border-color；

盒子阴影

display属性

display属性取值：

  • block块元素
  • inline行内元素
  • inline-block行内块元素
  • table以表格形式显示，类似于table元素
  • table-cell以表格单元格形式显示，类似于td元素
  • table-row以表格行形式显示，类似于tr元素
  • none隐藏元素

伸缩布局

1、flex：在主轴的缩放比例，不指定flex属性，则不参与伸缩分配；或指定min-width最小宽度、max-width最大宽度

2、flex-direction：调整主轴方向（默认为水平方向）；column 垂直排列、row 水平排列

3、justify-content：调整主轴对齐（水平对齐）

  • flex-start默认值；位于容器的开头
  • flex-end位于容器的结尾
  • center位于容器的中心
  • space-between位于各行之间留有空白的容器内
  • space-around位于各行之前、之间、之后都留有空白的容器内

4、align-items：调整侧轴对齐（垂直对齐）

  • stretch默认值；被拉伸以适应容器
  • center位于容器的中心
  • flex-start位于容器的开头
  • flex-end位于容器的结尾

5、flex-wrap：控制是否换行

  • nowrap默认值；不换行，收缩、强制一行内显示
  • wrap在必要的时候拆行或拆列
  • wrap-reverse在必要的时候拆行或拆列，但是以相反的顺序

6、flex-flow：排列方向（flex-direction）、换不换行（flex-wrap）的简写形式

7、align-content：多行垂直对齐方式齐
注意：必须对父元素设置自由盒属性display:flex;，并且设置排列方式为横向排列flex-direction:row;并且设置换行flex-wrap:wrap;这个属性的设置才会起作用

  • stretch默认值；被拉伸以适应容器
  • center位于容器的中心
  • flex-start位于容器的开头
  • flex-end位于容器的结尾
  • space-between位于各行之间留有空白的容器内
  • space-around位于各行之前、之间、之后都留有空白的容器内

8、order：排列顺序，从小到大
用整数值来定义排列顺序，数值小的排在前面；可以为负值；默认值是 0

浮动（float）

1、布局：标准流、浮动和定位

2、什么是浮动：元素的浮动是指设置了浮动属性的元素会脱离标准普通流的控制，移动到其父元素中指定位置的过程

  • left元素向左浮动
  • right元素向右浮动
  • none元素不浮动（默认值）

3、浮动脱离标准流，不占位置，会影响标准流；浮动只有左右浮动

4、清除浮动本质：

  • 清除浮动主要为了解决父级元素因为子级浮动引起内部高度为0 的问题
  • 把浮动的盒子圈到里面，让父盒子闭合出口和入口不让他们出来影响其他元素

5、清除浮动方法：

  • clear:both（多无意义的标签，结构化较差）
  • overflow:hidden（无法显示需要溢出的元素）
  • after伪元素

过渡（transition）

transition可以实现补间动画（过渡效果）

2D变形（transform）

transform是CSS3中具有颠覆性的特征之一，可以实现元素的位移、旋转、倾斜、缩放，甚至支持矩阵方式，配合过渡和即将学习的动画知识，可以取代大量之前只能靠Flash才可以实现的效果。

1、移动 translate(x, y)

  • translate(x,y)水平方向和垂直方向同时移动（也就是X轴和Y轴同时移动）
  • translateX(x)仅水平方向移动（X轴移动）
  • translateY(Y)仅垂直方向移动（Y轴移动）

2、缩放 scale(x, y)

  • scale(X,Y)使元素水平方向和垂直方向同时缩放（也就是X轴和Y轴同时缩放）
  • scaleX(x)元素仅水平方向缩放（X轴缩放）
  • scaleY(y)元素仅垂直方向缩放（Y轴缩放）

3、旋转 rotate(deg)
transform-origin可以调整元素转换变形的原点

4、倾斜 skew(deg, deg)

3D变形（transform）

1、rotateX() 沿着 x 立体旋转
2、rotateY()沿着y轴进行旋转
3、rotateZ()沿着z轴进行旋转
4、perspective透视
perspective 一般作为一个属性，设置给父元素，作用于所有3D转换的子元素
5、translateX(x)仅水平方向移动
6、translateY(y)仅垂直方向移动
7、translateZ(z)
8、translate3d(x,y,z)
其中，x和y可以是长度值，也可以是百分比，百分比是相对于其本身元素水平方向的宽度和垂直方向的高度和；z只能设置长度值
9、backface-visibility 属性定义当元素不面向屏幕时是否可见

动画（animation）

animation:动画名称 动画时间 运动曲线 何时开始 播放次数 是否反方向;
animation-iteration-count:infinite; 无限循环播放
animation-play-state:paused; 暂停动画”
