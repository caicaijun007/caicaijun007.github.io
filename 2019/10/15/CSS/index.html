总结CSS

发表于 2019-10-15 更新于 2019-10-17 分类于 web前端

CSS全称：Cascading Style Sheet（层叠样式表）

CSS特性

  • 继承性（子元素继承父元素的某些样式属性；不是所有属性都具有继承性；例如：padding、margin、border不具备继承性）

  • 层叠行（样式的覆盖；元素、属性、权重都相同，后者覆盖前者）

CSS优先级

行内样式>ID选择器>类选择器>标签选择器

不管是单个选择器还是多个选择器组合他们之间的优先级都可以通过上面这个公式来进行计算。我们将其称之为权重值。权重值越大，表示优先级越高。

样式冲突情况：

  • 引用方式冲突
  • 继承方式冲突
  • 指定样式冲突
  • 继承样式与指定样式冲突
  • ！important

层次选择器：

  • 相邻选择器 例如：a+b{}
