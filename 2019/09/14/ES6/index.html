ES6基础巩固与常用特性总结

ES6是强制开启严格模式

1、let 和 const 命令

  • let：只在声明代码块内有效(块级作用域)；不允许在相同作用域内，重复声明同一个变量；不会出现“变量提升”现象。
  • const：只在声明代码块内有效(块级作用域)；声明常量，一旦声明，其值不能修改。
    注意：数组、对象使用const定义常量，指针指向的是引用地址。

2、变量的解构赋值

  • 数组的解构赋值：按次序排列；如果结构不成功，变量值就等于undefined；undefined 和 null不能转为对象，因此会报错。

  • 对象解构赋值：变量必须与属性同名；可以嵌套结构；

3、字符串扩展

  • 模板字符串(字符串中嵌入变量)
  • contains(str)、startsWith(str)、endsWith(str)、repeat(重复次数)

9、Promise对象

promise对象的3个状态

  • pending: 初始化状态
  • fullfilled: 成功状态
  • rejected: 失败状态

10、Iterator（遍历器）

  • 数组、字符串、argument、Set、Map都内置Iterator就可以使用for of进行循环遍历

13、Module模块

export、import关键字

  • 使用export 关键字导出模块
  • 使用import 关键字引入模块
  • 使用export default 导出模块 (默认导出，不指定属性、方法名称)

14、数组、对象的深/浅拷贝

浅拷贝：拷贝的是数据的引用，修改拷贝后的数据会影响原数据

  • 直接赋值给一个变量
  • concat()
  • slice()
  • assign()

深拷贝：拷贝生成新数据，修改拷贝后的数据不会影响原数据

15、Set、Map容器

Set：无序不可重复的多个value的集合体 （数组的定义）
作用：数组去重

  • size
  • Set([1, ‘a’, 34])
  • add(value)
  • has(value)
  • delete(value)
  • clear()

Map: 无序key不可重复的多个key-value的集合体

  • size
  • Map([[‘a’, ‘first’], [‘b’, ‘second’]])
  • set(key, value)
  • get(key)
  • has(key)
  • delete(key)
  • clear()
