ES5常用特性总结

1、JSON对象

  • JS(数组/对象)转为JSON(数组/对象)使用JSON.stringify()
  • JSON(数组/对象)转为JS(数组/对象)使用JSON.parse()

2、Array数组

var arr = new Array();

  • arr.length：数组长度

  • arr.indexOf(value)：数组是否包含字符串值，否则返回-1，判断字符串是否存在数组中

  • arr.forEach(function(item, index){})：一般遍历数组

  • arr.map(function(item, index){})：遍历数组返回一个新的数组

  • arr.filter(function(item, index){ })：遍历且过滤出一个新的数组

  • typeof返回数据类型：String、Number、Object、Function、Boolean、Undefined
    如何判断Array、Null数据类型？

  • argument伪数组：不具有数组的一般方法

3、Object对象

var obj = { name: ‘a’, age: 13 };

  • Object.defineProperties(obj, param)：为指定对象定义扩展多个属性

4、function方法

var obj = new Object();
var obj2 = new Object();

  • obj.call(obj2, arg1, arg2,…)
  • obj.apply(obj2, [arg1, arg2,…])
  • obj.bind(obj2, arg1, arg2,…)

相同点：

  • 改变this指向的对象，第一个参数都为指向的原型对象

区别：

  • call()、bind()从第二个参数开始传形参，以此传入；
  • apply()第二参数必须为数组，数组为传递的形参。
  • call()、apply()会立即执行当前函数；bind()不会立即执行当前函数。
